# Homebrew Tap

Homebrew formulae for Aaron Faby projects.

## Install

Install `icloud` with:

```sh
brew install AaronFaby/tap/icloud
```

Or tap this repository first:

```sh
brew tap AaronFaby/tap
brew install icloud
```

## Formulae

- `icloud`: JSON-first CLI for iCloud Mail, Calendar, and Contacts automation through documented protocols.

## Maintenance

Formula changes are validated with Homebrew test-bot on pull requests.
The `icloud-cli` release workflow can dispatch `bump-icloud.yml` to open a formula update pull request after a new `v*` release.
